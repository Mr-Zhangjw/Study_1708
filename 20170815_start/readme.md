# 1. 行内文本标签

修饰文本（文字）的标签,全是成对出现

    b  字体加粗
    i  斜字体
    u  定义下划线
    del  
    sub 
    sup

# 2. 标题

# 3. 段落

    p

段落标签p里面不支持换行，但是段落之间，默认换行

# 4. 换行

    br  自结束标签

常用于文字内部，不产生行距

# 5. 水平线

    hr

# 6. 分区元素

    div     块元素    不相当于文字，“盒”   

        1. 从上往下排列
        2. 高宽可以直接生效
        3. 默认宽度 100%

    span    行内元素  相当于文字

        1. 默认从左往右
        2. 高宽无法直接生效
        3. 默认宽度 按内容尺寸显示
